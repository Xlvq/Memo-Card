# Документация к приложению "Викторина"

## Общее описание
Это приложение представляет собой викторину с вопросами и вариантами ответов, реализованную с использованием PyQt5. Программа состоит из главного меню и окна викторины, позволяет проходить тесты, сохранять статистику и просматривать результаты.

## Структура проекта

### Основные классы
1. **Question** - класс для хранения информации о вопросе:
   - `question` - текст вопроса
   - `right_answer` - правильный ответ
   - `wrong1`, `wrong2`, `wrong3` - неправильные варианты ответов

### Основные окна
1. **main_menu** - главное меню приложения
2. **quiz_win** - окно викторины с вопросами

### Элементы интерфейса
- **Главное меню**:
  - Кнопка "Начать новую игру"
  - Кнопка "Просмотр статистики"
  - Кнопка "Выйти"

- **Окно викторины**:
  - Поле с вопросом
  - Группа радиокнопок с вариантами ответов
  - Кнопка "Ответить"
  - Блок результатов с информацией о правильности ответа
  - Кнопка "Следующий вопрос"
  - Кнопка "Завершить"

## Функционал

### Работа с вопросами
1. **load_questions()** - загружает вопросы из файла questions.json
   - Формат файла: JSON-массив объектов с полями:
     - `question` - текст вопроса
     - `right_answer` - правильный ответ
     - `wrong_answers` - массив из 3 неправильных ответов

2. **ask(q: Question)** - отображает вопрос и варианты ответов
   - Перемешивает варианты ответов случайным образом
   - Устанавливает текст вопроса и вариантов ответов

### Работа со статистикой
1. **load_stats()** - загружает статистику из файла stats.json
2. **save_stats()** - сохраняет статистику в файл stats.json
3. **calculate_rating()** - вычисляет процент правильных ответов
4. **print_stats()** - выводит статистику в консоль
5. **show_stats()** - отображает статистику в диалоговом окне

### Логика викторины
1. **next_question()** - выбирает случайный вопрос и вызывает ask()
2. **check_answer()** - проверяет выбранный ответ
3. **show_correct(text)** - показывает результат ответа
4. **show_question()** - показывает новый вопрос
5. **click_ok()** - обработчик кнопки "Ответить"/"Следующий вопрос"

### Управление приложением
1. **start_game()** - запускает викторину
2. **exit_game()** - завершает игру с показом статистики
3. **exit()** - завершает приложение

## Форматы файлов

### questions.json
```json
[
    {
        "question": "Текст вопроса",
        "right_answer": "Правильный ответ",
        "wrong_answers": ["Неправильный1", "Неправильный2", "Неправильный3"]
    },
    ...
]
```

### stats.json
```json
{
    "total_questions": 0,
    "correct_answers": 0
}
```

## Использование
1. Запустите приложение
2. В главном меню выберите:
   - "Начать новую игру" - для начала викторины
   - "Просмотр статистики" - для просмотра общей статистики
   - "Выйти" - для завершения работы

3. В режиме викторины:
   - Выберите вариант ответа и нажмите "Ответить"
   - Для перехода к следующему вопросу нажмите "Следующий вопрос"
   - Для завершения нажмите "Завершить"
